# ScanPoint

Scan POI's around the world!

## Start the Api
First, get the dependencies
```bash
pip install -r requirements.txt
```
Then, start the application
```bash
uvicorn main:app --reload --host 192.168.1.90 --port 8000
```

## Start the App
First, get the dependencies
```bash
flutter pub get
```
Then, start the application
```bash
flutter run
```
